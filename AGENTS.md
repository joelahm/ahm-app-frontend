# Project Conventions

These conventions are mandatory for this project unless explicitly overridden by the user in a specific task.

1. UI library
- Always use HeroUI components for UI implementation.

2. Component design
- Always build reusable components.
- Prefer shared components in `components/` over page-only one-off markup.

3. Forms and validation
- Use `react-hook-form` for form state handling.
- Use Yup schemas for form validation.

4. Icons
- Use Lucide icons whenever an icon is needed.
